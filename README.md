## Usage

```
cargo run
```

### Controls
 - Move right and left: `Q - D / ← - →`
 - Rotate right: `Z / ↑ / C`
 - Rotate left: `X`
 - Drop: `S / ↓`
 - Hard-drop: `<Space>`
 - Hold: `H`
